# Speakly Desktop Releases

Official binary releases for Speakly Desktop App.

## Download

Visit [speakly.app](https://speakly.app) to download the latest version.

Or go to the [Releases](https://github.com/rodolfobortolin/speakly-releases/releases) page and download the appropriate file for your system:

| Platform | File |
|----------|------|
| macOS (Apple Silicon) | `Speakly-*-arm64.dmg` |
| macOS (Intel) | `Speakly-*.dmg` |

## About

Speakly is a voice-to-text transcription app with AI-powered formatting.

> **Note:** This repository contains only binary releases. The source code is not available here.
